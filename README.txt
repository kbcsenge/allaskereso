Szegedi Tudományegyetem Természettudományi és Informatikai Kar Adatbázis alapú rendszerek gyakorlatára készített program.
Társszerzők: Kollár Laura, Kalmár Krisztián

Fő funlcióK
  -bejelentkezés/regisztráció
  -álláshirdetés feladása
  -állásra jelentkezés
  -állás értékelése
  -egyszerre több, idegennyelvű életrajz feltöltése
  -álláshirdetések összetett szűrése
 
 A program Oracle SQL Developerben készült adatbázisra csatlakozik. Működése során triggereket és PL/SQL tárolt eljárásokat is használ.
